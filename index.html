How to Open HTML Files Offline

  1. Go to your Files app.
  2. Find and open the HTML file.
  3. If blocked, go to browser History.
  4. Find the file → 3-dot menu → More from this site.
  5. Add ?id=800 at the end.
  6. Change start to file:///
